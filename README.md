## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/nest-market.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run database migrations
npx prisma migrate dev

# Start development server
npm run dev
```
